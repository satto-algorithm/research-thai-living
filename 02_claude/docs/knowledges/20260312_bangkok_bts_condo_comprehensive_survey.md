## 契約に関する一般的な注意事項

| 項目 | 一般的な条件 |
|------|------------|
| **最低契約期間** | 1年（12ヶ月）が標準 |
| **デポジット** | 家賃2ヶ月分が標準（一部物件は1ヶ月） |
| **前払い家賃** | 入居時に1ヶ月分前払い |
| **初期費用合計** | 家賃3ヶ月分（デポジット2ヶ月＋前払い1ヶ月） |
| **電気代** | 家賃別途（MEA直接契約で約5-7 THB/ユニット） |
| **水道代** | 家賃別途（約18-25 THB/ユニット） |
| **インターネット** | 別途契約（True/AIS/3BB等、月500-800 THB程度） |
| **共益費** | 通常オーナー負担（賃借人は不要） |
| **契約更新** | 通常は年次更新、家賃交渉可能 |
